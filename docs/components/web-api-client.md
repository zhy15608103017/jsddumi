# web-api-client

## 背景

基于 umi-request库(v1.4.0) 进行二次封装  
主要是在请求头上增加了接口调用的认证信息(authorization)  
具体参数可以直接看umi-request库的文档

## 使用方法

```bash
$ npm install --save @jusda-tools/web-api-client --registry http://nexus.jusda.int/verdaccio/
```
